# Contributing to gov-search-app

## Getting Started
- Fork the repo and create your branch from `main`.
- Install dependencies in both `backend` and `frontend`.
- Run the servers locally for development.

## Pull Requests
- Provide clear PR titles and descriptions.
- Reference related issues, if any.
- One feature/bugfix/change per PR recommended.

## Code Style
- Use consistent formatting (Prettier recommended for JS).
- Prefer functional components and hooks in React.
- Node/Express: Use async/await, keep error handling clear.

## Tests
- Include or update simple tests if possible.

## Other Guidelines
- Be respectful in discussions/reviews!
- All contributors must follow the Code of Conduct.
